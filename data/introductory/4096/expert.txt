Write a function called that takes a string of parentheses, and determines if the order of the parentheses is valid. The function should return true if the string is valid, and false if it's invalid. 0 <= input.length <= 100.

Examples:
Input:
"()"
Output:
true

Input:
")(()))"
Output:
false

Input:
"("
Output:
false

Input:
"(())((()())())"
Output:
true
