 Given a number, you need to find number of ways to make it sum of numbers. 
Examples ### Basic ```python exp_sum(1) # 1 exp_sum(2) # 2  -> 1+1 , 2 exp_sum(3) # 3 -> 1+1+1, 1+2, 3 exp_sum(4) # 5 -> 1+1+1+1, 1+1+2, 1+3, 2+2, 4 exp_sum(5) # 7 -> 1+1+1+1+1, 1+1+1+2, 1+1+3, 1+2+2, 1+4, 5, 2+3 exp_sum(10) # 42 ``` ### Explosive ```python exp_sum(50) # 204226 exp_sum(80) # 15796476 exp_sum(100) # 190569292 ``` See [here](http://www.numericana.com/data/partition.htm) for more examples.
